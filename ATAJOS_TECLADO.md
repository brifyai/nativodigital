# ⌨️ Atajos de Teclado - Nativo Digital

## Navegación General

| Atajo | Acción |
|-------|--------|
| `Ctrl + N` / `⌘ + N` | Crear nuevo chat |
| `Ctrl + B` / `⌘ + B` | Mostrar/Ocultar sidebar |
| `Ctrl + H` / `⌘ + H` | Abrir centro de ayuda |
| `Ctrl + /` / `⌘ + /` | Enfocar campo de entrada |
| `Esc` | Cerrar modales abiertos |

## Escritura de Mensajes

| Atajo | Acción |
|-------|--------|
| `Enter` | Enviar mensaje |
| `Shift + Enter` | Nueva línea en el mensaje |

## Accesibilidad

- **Navegación por Tab**: Todos los elementos interactivos son accesibles con Tab
- **ARIA Labels**: Todos los botones tienen etiquetas descriptivas para lectores de pantalla
- **Alto Contraste**: Disponible en Ajustes → Accesibilidad
- **Indicadores Visuales**: Feedback claro para todas las acciones

## Consejos Pro

1. **Usa `Ctrl + N`** frecuentemente para organizar temas en diferentes chats
2. **`Ctrl + /`** es perfecto para volver rápido al input sin usar el mouse
3. **`Esc`** cierra cualquier modal, muy útil para navegación rápida
4. **Alto Contraste** mejora la legibilidad en ambientes con mucha luz

## Soporte de Lectores de Pantalla

Nativo Digital es compatible con:
- NVDA (Windows)
- JAWS (Windows)
- VoiceOver (macOS/iOS)
- TalkBack (Android)

Todos los elementos interactivos tienen etiquetas ARIA apropiadas.
